Detailed Description

Base class for a tree.

Warning
Unless you need MAXIMUM PERFORMANCE, your are better using its derived VS_XML_NS::Tree

This one does not wrap returned XML entities nor string views, so you are left on your own to handle them, but you might gain few points in complex pipelines.
